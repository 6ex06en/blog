
Удаление поста.
  Когда авторизовался собственник поста
    после удаления статьи
      должен быть редирект на главную страницу
      пост не должен отображаться
      пост должен быть удален из БД
  Когда авторизовался другой пользователь
    при попытке удаления поста
      должен быть редирект на главную странцу
      пост не должен быть удален
  Неавторизованный пользователь
    при попытке удаления поста
      пост не должен быть удален

Редактирование поста.
  Когда авторизовался собственник поста.
    Когда пост отредактирован
      должна отрендериться страница редактирования поста
      должно появиться flash сообщение
      содержание поста должно измениться
    Когда пост отредактировать не удалось
      должна отрендериться страница редактирования поста
      должно появиться сообщение об ошибке
      имя поста не должно измениться
  Неавторизованный пользователь
    не сможет зайти на страницу редактирования поста
      должен увидеть flash сообщение
      должен быть переадресован на root
  Не собственник поста пользователь
    не сможет зайти на страницу редактирования чужого поста
      должен быть переадресован на root

Создание поста.
  Когда успешно создан
    должна отрендериться главная страница
    должно появиться flash сообщение
    пост должен отразиться на странице
  Когда создать не удалось
    должна отрендериться страница создания поста
    должны появиться сообщения с описанием ошибки
    на главной странице не должно быть поста
